# 🎮 猜数字游戏（Python实战）

## 🧠 游戏规则

电脑随机生成一个 1~10 的数字  
你来猜，如果猜对就赢

---

## 📌 完整代码

```python
import random

# 生成随机数
num = random.randint(1, 10)

while True:
    guess = int(input("请输入你猜的数字（1-10）："))

    if guess == num:
        print("🎉 恭喜你猜对了！")
        break
    elif guess > num:
        print("太大了⬆️")
    else:
        print("太小了⬇️")
```

## 💡 你学到了什么

- random：生成随机数
- input：用户输入
- if：判断逻辑
- while：循环
- break：停止循环
