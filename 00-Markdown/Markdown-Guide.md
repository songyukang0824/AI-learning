# 📌 什么是 Markdown？

Markdown 是一种**轻量级标记语言**。

你可以理解成：

> **Markdown = 不用鼠标的 Word。**

它使用一些简单的符号，就可以完成排版。

例如：

```markdown
# 一级标题

## 二级标题

**加粗**

- 列表
```

Markdown 被广泛应用于：

- GitHub
- Gitee
- Typora
- Obsidian
- VS Code
- 博客系统

以后学习 AI、Python、写项目文档都会用到。

# 📌 Markdown 编写规范（★★★★★）

> 💡 建议养成规范，以后写出来的文档会非常舒服。

## 1️⃣ 一篇文章只使用一个一级标题

推荐：

```markdown
# AI 学习日志
```

不要：

```markdown
# 今日目标

# 今日学习

# 今日总结
```

一篇文章一个一级标题即可。

---

## 2️⃣ 标题下面空一行

推荐：

```markdown
## 今日学习

今天学习了 GitHub。
```

不要：

```markdown
## 今日学习
今天学习了 GitHub。
```

---

## 3️⃣ 每个模块之间空一行

推荐：

```markdown
## 今日学习

......

---

## 今日总结
```

这样阅读体验更舒服。

---

## 4️⃣ 每个模块之间建议使用分割线

输入：

```markdown
---
```

效果：

---

这样文章层次会更清晰。

---

## 5️⃣ 使用列表代替大段文字

推荐：

```markdown
- GitHub
- Markdown
- AI
```

不要：

```
GitHub、Markdown、AI……
```

列表更容易阅读。

# 📌 Markdown 基础语法

---

# ① 标题

## 一级标题

输入：

```markdown
# 一级标题
```

效果：

# 一级标题

---

## 二级标题

输入：

```markdown
## 二级标题
```

效果：

## 二级标题

---

## 三级标题

输入：

```markdown
### 三级标题
```

效果：

### 三级标题

> 💡 一般情况下，三级标题已经够用了。

---

# ② 普通文字

直接输入即可。

例如：

```markdown
今天开始学习 AI。
```

---

# ③ 加粗（★★★★★）

输入：

```markdown
**重点内容**
```

效果：

**重点内容**

适用于：

- 重点知识
- 重要提醒
- 面试重点

---

# ④ 斜体

输入：

```markdown
*说明文字*
```

效果：

*说明文字*

一般较少使用。

---

# ⑤ 删除线

输入：

```markdown
~~已经完成~~
```

效果：

~~已经完成~~

---

# ⑥ 无序列表（★★★★★）

输入：

```markdown
- GitHub
- Python
- AI
```

效果：

- GitHub
- Python
- AI

---

# ⑦ 有序列表

输入：

```markdown
1. 学 AI
2. 学 Python
3. 做项目
```

效果：

1. 学 AI
2. 学 Python
3. 做项目

---

# ⑧ 引用

输入：

```markdown
> 今天最大的收获
```

效果：

> 今天最大的收获

一般用于：

- 总结
- 名人名言
- 学习心得

---

# ⑨ 分割线（★★★★★）

输入：

```markdown
---
```

效果：

---

建议：

每个模块之间使用一次。

---

# ⑩ 行内代码

输入：

```markdown
`print()`
```

效果：

`print()`

适合：

函数名

变量名

文件名

例如：

```markdown
`README.md`
```
